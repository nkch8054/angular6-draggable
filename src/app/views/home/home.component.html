Angular2-Draggable

angular2-draggable has Angular(ver >= 4.x) directives that make the DOM element draggable and resizable

Documents Github

| Directives | Demo | | ---------- | ---- | |`ngDraggable` | [demo](#/draggable/usage/basic) | |`ngResizable` | [demo](#/resizable/default) |

Installation

```bash # if you use npm npm install angular2-draggable --save # or if you use yarn yarn add angular2-draggable ```

Import

```diff + import {{'{'}} AngularDraggableModule } from 'angular2-draggable'; @NgModule({{'{'}} imports: [ ..., + AngularDraggableModule ], }) export class AppModule {{'{'}} } ```
